# ADR-002: 1.77" 横屏 IPS 显示

## 状态

已接受

## 背景

8HP 只能竖屏 128×160；10HP 宽 50.3 mm 可横屏 160×128，频谱横向分辨率更好。

## 决策

- 面板 **10HP Core**
- **1.77" IPS**，ST7735，**横屏** 160×128
- Outline 43.8 × 34 mm

## 理由

- 频谱条横向更宽，产品核心差异化
- 比 O_C 小 OLED 可读性大幅提升
- 横屏比竖屏省 ~10 mm 面板高度给控件/插孔

## 后果

- 屏占满宽度，无法做 O_C 式屏侧小键
- UI 布局按宽屏重设计
